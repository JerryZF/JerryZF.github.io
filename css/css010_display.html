Block

  • Take the entire line
  • It has no height
  • You may set dimensions

Inline

  • No width, it depends on the contents
  • Height depends on the fonts' height
  • Cannot set dimensions

Inline block

  • No width, No height.
  • Can set dimensions

None

  • Do Not display
  • Do Not take space

Visibility

  • Not display, but may set dimensions
